# Git Workflow Task Manager

A simple, client-side, one-page Kanban board designed to track tasks through a standard Git lifecycle. 

## Features
- **Zero dependencies:** Built entirely with vanilla HTML, CSS, and JavaScript.
- **Drag-and-Drop:** Move tasks seamlessly between columns.
- **Git-Aligned Columns:**
  - `Issues`: Tasks that are ready to be picked up.
  - `Active Branches`: Tasks currently being worked on in a local branch.
  - `Pull Requests`: Code pushed and awaiting review.
  - `Merged`: Code successfully merged into the main branch.

## How to Run
Because there is no backend or build step, simply open `index.html` in any modern web browser to use the app.
